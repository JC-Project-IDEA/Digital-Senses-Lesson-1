# Digital Senses Lesson 1 : P5.js 基礎教學及基本編程技巧


甚麼是 P5.js?
p5.js是一個用作創意編碼的JavaScript 函式庫，重點是讓藝術家、設計師、 教育工作者、初學者和其他任何人都能獲得編碼的機會和包容!p5.js是免費和開源的，因為我們相信軟件和學習軟件的工具應該是人人都能獲得的。
